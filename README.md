# sectioncomment

Creates an 80 character width comment padded with hyphens, similar to how 
comments are shown in Hadley's style guide (http://adv-r.had.co.nz/Style.html)

## System Requirements

## Installation

```r
devtools::install_github("ReportMort/sectioncomment")
```
